## Zadanie 2




Stwórz obiekt ```slider```, taki jak poniżej:

```js
const slider = {
  type: "infinite",
  numberOfItems: 10,
  center: true,
  autoStart: true
}
```

Podstaw do zmiennych ```type``` oraz ```autoStart``` odpowiednie wartości z obiektu ```slider```.
